== Description ==

CleanLab is one of the most easy to use WordPress themes developed by ThemeFuzz. It includes more than 60 pagebuilder elements, each packed with it's own options and different styles that will help you create your entire site in minutes.

Features:

* Frontend Pagebuilder
* WPML & translation ready
* WooCommerce ready with additional WooCommerce features
* Responsive design
* 1 click demo data install
* Unlimited colors
* 5 Header layouts
* Icon font uploader
* Mega menu
* Unlimited sidebars
* Coming Soon features
